RAG
—
How many of the last conversations to give to the AI for context. Set value as -1 for all the previous messages in this session. Important: Having more context may take more time to generate responses.
Collection name for the vector DB. Consider it as a folder with different folders having different subjects. You have to give the name for a new folder to create or, if already existing, use.
The LLM model to answer based on the retrived context. The default model (deepseek-r1:1.5b) is sufficient for the task, however, if you want to use another open source model, you would have to download it. Refer to README.md on how to download models.
